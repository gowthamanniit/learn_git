	open github and create repositary

step 2:
	code --> http : copy link

step 3:
	open cmd propmt or terminal in vs code

	d:\...\....> cd\

	d:\>git clone pastlink

	ex:

	d:\>git clone https://github.com/gowthamanniit/reactrepo.git

	note: check your local disk repositary folder and .git file.
step 4:
	In vs code --> file --> open folder -->git folder select
						(ex: reactrepo)

step 5:
	d:\>cd reactrepo (if not found)
	d:\reactrepo>npm install -g create-react-app
	d:\reactrepo>npx create-react-app . 
	(all library files - node_modules, package.json... src public)
	
	d:\reactrepo>if error web-vitals
	d:\reactrepo>npm install web-vitals --save-dev
	d:\reactrepo>npm start
